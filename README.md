#### 오늘 공부한 내용
- 웹 : request와 response의 연속
- 웹이 해줘야 할 기능
    - Create
    - Read
    - Update
    - Delete

- 웹 프로세싱
    - url로 request
    - model - views - templates
- migrate
    - 파이썬의 데이터구조가 DB의 데이터구조와 다르다
    - 이를 구조에 맞게 고쳐주는 행위
    - python manage.py makemigraions -> python manage.py migrate
- html 나누기
    - extends 태그 사용하기
    - base.html -> 아들 html들
